**actions/actions bonus/réactions/ressources communes  au joueurs :**  
**PV** : quantité de dégâts pouvant être encaissés, si 0 \-\> état critique  
**vitesse** de déplacement (5m \+SPD) : permet *se déplacer* d’une certaine distance.  
(note : il est possible de se déplacer plusieurs fois par tour, tant qu’il reste de la *vitesse*)

**dash/course** (action)  (déplacement \* 2\)    
**help** (action) : permet notamment de retirer certaines conditions (état critique inclus)  
**attaque** (action) : attaque avec l’arme en main  
**lancer** (action) : lancer n’importe quel objet/créature à portée ou dans l’inventaire  
(note : certains objets consommables peuvent être lancé pour appliquer leur effet à distance)

**attaque d’opportunité** (réaction) : si A s’éloigne de B, B peut faire une action gratuite contre A immédiatement.  Une différence de stats en vitesse peut prévenir cette réaction.
