termes spécifiques :

**jet/check** : d20 pour savoir si une action réussit, on y ajoute généralement un modificateur dû aux stats.

**jet d’attaque** : pour savoir si une attaque touche, ou si l’effet d’un sort s’applique

**jet de défense** : terme générique pour *parade* (DEX), *blocage* (STR) ou *esquive* (SPD)  
(peut aussi inclure des compétences spécifiques aux classes)

- esquive : jet SPD vs to hit adverse, marche vs AOE (selon cas)  
- parade : jet DEX vs to hit adverse, marche pas vs AOE  
- blocage : jet STR vs dmg adverse, marche vs AOE

(certaines attaques ne peuvent être que esquivées)

**save/jet de sauvegarde :** jet pour résister à un effet négatif, typiquement basé sur une stat spécifique

**jet de précision / coup précis** : action réservée aux classes martiales,   
jet (PRE) supplémentaire dans le cas où une attaque touche. Cible une AC différente nommée AC2. Permet au joueur d’appliquer des effets supplémentaires.

**proficiency/familiarité** : capacité d’un perso à utiliser tels ou tels objets/équipements,   
pénalités dans le cas où un perso ne les respecte pas.

abréviations (pour lecture des fiches) :  
**ordre des stats :** STR DEX SPD PRE CHAR INT POW  
(POW absent pour les classes non magiques)

atk : attaque  
dmg : damage  
mv : déplacement (move)

**PV** : points de vie  
**PR** : points de rage (barbare)  
**PM** : points de mana (mages)  
**PF** : points de foi (paladin/prêtres)
